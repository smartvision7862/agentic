Webhook Setup
1. Run the local server: npm start in /content-os
2. Expose it: npx localtunnel --port 3950
3. In Twilio Console → WhatsApp Sandbox → set webhook URL to:
{your-tunnel-url}/api/webhooks/whatsapp
